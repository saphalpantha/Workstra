# Workstra

A simple and responsive landing page built with clean HTML and CSS. This project was created as a practice project to explore layout design, responsive styling, and consistent code formatting.

---

## Description

Workstra is a demo landing page developed using HTML, CSS, and modern design principles. It focuses on simplicity, responsive layout, and readability. The project serves as a foundation for experimenting with front-end structure, styling, and deployment workflows.

---

## Features

- Responsive layout for all screen sizes  
- Clean and minimal UI design  
- Organized code structure  
- Prettier configuration for consistent formatting  
- Ready-to-deploy static website
